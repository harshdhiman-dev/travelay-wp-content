== Description ==

Top Deals on Flights is a Gutenberg block plugin that displays flight deals in a beautiful grid layout. It integrates with the Amadeus API (via Flight Offers DigitalSilk helpers) to fetch real-time flight data.

== Features ==

* Gutenberg block integration
* Amadeus API integration
* Customizable grid columns (desktop, tablet, mobile)
* Savings badge display
* Responsive design
* Fallback to sample data if API unavailable

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Ensure Advanced Custom Fields Pro is installed and active
4. Add the "Top Deals on Flights" block to any page
